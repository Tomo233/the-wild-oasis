        <h2 style="color: #555;">Features</h2>
        <ul style="list-style: disc; padding-left: 20px;">
            <li>User Authentication: Secure user authentication system implemented for ensuring user privacy and data security.</li>
            <li>Booking Management: Efficiently manage cabin bookings, including reservations, cancellations, and modifications.</li>
            <li>Advanced Compound Component Pattern: Utilizes advanced compound component patterns for building reusable, flexible, and composable UI components.</li>
            <li>Responsive Design: Ensures optimal user experience across various devices and screen sizes.</li>
            <li>Charts Integration: Visualize booking statistics and trends with integrated charts for data analysis.</li>
            <li>Dark Mode: Offers a sleek and modern dark mode option for improved usability in low-light environments.</li>
            <li>Professional Application Planning and Development: Developed with a focus on professional planning, design, and development practices for robust and scalable application architecture.</li>
        </ul>
        <h2 style="color: #555;">Technologies Used</h2>
        <ul style="list-style: disc; padding-left: 20px;">
            <li>React Query: For efficient data fetching and caching, ensuring smooth user interactions.</li>
            <li>Styled Components: Dynamically style React components with scoped styles for enhanced modularity and maintainability.</li>
            <li>React Hook Form: Simplifies form management with React hooks, optimizing form performance and user experience.</li>
            <li>Supabase: Utilized as the backend for managing data storage, authentication, and real-time updates.</li>
            <li>Advanced Compound Component Pattern: Implemented for building flexible and reusable UI components, enhancing code maintainability and scalability.</li>
        </ul>
